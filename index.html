Infix notation to PharmML math online converter

This web service converts a mathematical expression in the usual infix notation into the corresponding PharmML markup. PharmML is a markup for pharmacometrics documented at http://pharmml.org/ and developed within the DDMoRe consortium.

Converter version 0.6.x; the output should be PharmML 0.6-compliant.

Quickstart

This is an online version of the converter. For details and source code see the infix2pharmml project page at SourceForge.

The converter works in either single-statement or stand-alone model modes.

The single-statement mode converts various types of statements, one at a time. The user is responsible of combining them and adding boilerplate code to get a fully-functional (e.g., XML-valid) model.

The stand-alone mode creates a mostly ready-to-run model (only encompassing structural equations). In this mode, multiple statements can be provided, separated by semicolons. Statements are restricted to variable and derivative variable definitions. Stand-alone mode is intended to jump-start development, and it is not a full-fledged modeling tool.

For a full list of the converter's features and limitations, see the project's README.
